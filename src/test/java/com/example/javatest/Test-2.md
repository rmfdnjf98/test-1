## 네트워크 & 소켓 프로그래밍 서술

### TCP와 UDP의 차이점을 OSI 모델 관정에서 비교 설명하고, Java에서 각 프로토콜을 사용한 서버-클라이언트 통신 구현 방법을 서술하시오

- TCP와 UDP는 OSI 7계층의 전송계층에 위치하며 데이터를 전송하기 위하여 사용됩니다. TCP는 3-way handshake롤 데이터를 주고 받으며 신뢰성을 확보하고 UDP는 무작위로 데이터를 전송하여 신뢰성을 확보하기 어렵다

서버는 클라이언트의 요청으로

### Java 소켓 프로그래밍에서 멀티스레드 서버를 구현하는 방법과 ExecutorService를 활용한
